# 02 Pokemon API

This is a beginner FastAPI project.

The goal is to build a small Pokemon API step by step.

## What This Project Will Teach

- FastAPI basics
- REST API routes
- JSON responses
- path parameters
- query parameters
- request bodies
- Pydantic models
- CRUD actions

## Run The API

Install requirements:

```bash
pip install -r requirements.txt
```
