Welcome to BinaryBeasts

BinaryBeasts is a beginner-friendly Flask app that helps students learn how computers represent information using data.

Computers do not understand words, pictures, colours, or sounds the same way humans do. They store and process information using numbers, patterns, and binary values.

What is Data Representation?

Data representation is the way computers store information so it can be processed, saved, displayed, and shared.

In this project, you will explore how different types of information can be represented using simple values.

  • Numbers can be represented in binary
  • Text can be represented using character codes
  • Colours can be represented using RGB values
  • Images can be represented using pixels

Learning Tools

This project will be built step by step. Each tool teaches a different data representation concept.

  • Decimal to Binary Converter: Turn normal numbers into binary. Open tool
  • Binary to Decimal Converter: Turn binary values back into normal numbers. Open tool
  • Text to Binary Converter: Explore how letters can become binary. Open tool
  • RGB Colour Lab: Change red, green, and blue values to create colours. Open tool
  • Pixel Beast Activity: Use data values to build a tiny pixel creature. Open tool

Key Vocabulary

  • Bit: A single binary digit, either 0 or 1.
  • Byte: A group of 8 bits.
  • Binary: A number system that only uses 0 and 1.
  • Pixel: A tiny square of colour used to make digital images.
  • RGB: A colour system using red, green, and blue values.

Student Challenge

As you build this project, think about this question:

If computers only understand data, how do they show us words, colours, pictures, and games?
