# 05 Canteen Decision Checker

This project checks if the user can afford an item.

The program makes a decision based on the money left.

## What You Learn

- `if`
- `elif`
- `else`
- comparisons
- decisions in code

## Run It

Use:

`python3 main.py`

## Task

Run the program three times.

Try having enough money, exact money, and not enough money.
